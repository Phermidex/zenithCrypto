# **App Name**: Zenith Crypto Wallet

## Core Features:

- Credit Card Management: Add, update, and manage credit cards securely via Stripe integration.
- Crypto Purchase: Execute cryptocurrency purchases using stored credit cards.
- Wallet Balance Display: Display current cryptocurrency balances for each supported crypto.
- Market Data Display: Show basic market price data for available cryptocurrencies.

## Style Guidelines:

- Background color: Dark grey (#121212) to support a dark mode theme.
- Primary color: Deep green (#4CAF50) for main interactive elements.
- Accent color: Gold (#FFC107) for highlights and key information display.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral look.
- Use minimalist, vector-based icons in gold (#FFC107) for navigation and actions.
- Clean, card-based layout for displaying wallet balances and market data.
- Subtle transitions and animations for user interactions (e.g., loading indicators, confirmation messages).
